# GitJobs

### Description

Gets list of jobs from GitHub API and display it with date sections

JobsViewController -  implemented UI by IB
JobDetailsViewController -  implemented UI by code

### Dependencies

* ObjectMapper = 4.2.0;
* Realm = 10.7.2;
* RealmDatabase = 10.5.5;
* RxdataSources = 5.0.1;
* RxSwift = 6.1.0;
